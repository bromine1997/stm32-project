## 📚 Practice

기본 기능들을 하나씩 익히는 단계입니다. 약 2주간 진행 예정.

| 주제 | 상태 | 설명 |
|------|------|------|
| GPIO | ✅ 진행중 | 디지털 입출력, LED 제어, 버튼 인터럽트 |
| Timer Interrupt | 🔜 예정 | 타이머 기반 주기 인터럽트 |
| PWM | 🔜 예정 | 듀티 사이클 제어, 모터/LED 밝기 조절 |
| UART | 🔜 예정 | 시리얼 통신, 디버그 출력 |
| 추가 예정 | 🔜 | I2C, SPI 등 필요에 따라 추가 |
